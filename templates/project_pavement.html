Pavement Distress Detection and Quantification

August 2023 - April 2024 IIT Kharagpur | Prof. K K Kuna

Project Overview

This project focused on developing an automated system for the localization, classification, and quantification of various pavement distresses using advanced computer vision techniques. The goal was to create a robust solution that could identify different types of road damage from images and video footage, providing accurate measurements and assessments that could be used for infrastructure maintenance planning.

Technologies Used

YOLOv8 U-Net Python OpenCV TensorFlow Computer Vision Graph Traversal Algorithms

Key Features

  • Real-time Distress Detection

    Implemented YOLOv8 for effective classification of pavement distresses, improving the accuracy of identifying various types of damage in real-time video analysis.

  • Semantic Segmentation

    Applied U-Net for semantic segmentation to extract masked pavements, enabling precise identification of distress boundaries.

  • Distress Quantification

    Employed graph traversal algorithms to measure and quantify detected distresses, providing detailed metrics on length, width, area, and severity.

  • Performance Improvements

    Enhanced real-time processing speed by 40% through optimization of the detection pipeline while maintaining high accuracy.

Results and Impact

The system achieved an accuracy rate of 87% in detecting and quantifying pavement distresses, significantly improving upon previous methods. The real-time processing capabilities make it suitable for integration with mobile inspection vehicles, enabling faster and more comprehensive road condition assessments. This technology helps highway authorities prioritize maintenance activities and allocate resources more effectively.
