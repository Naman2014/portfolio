Automated Question Clustering System

September 2023 - November 2023 IIT Kharagpur | Prof. Mrigank Sharad

Project Overview

The Automated Question Clustering System was developed to organize and manage student questionnaires more efficiently. In educational settings, instructors often receive similar questions from different students, making it challenging to address each one individually. This project aimed to develop an automated system that could group similar questions together, allowing instructors to provide comprehensive answers to common themes and improve the efficiency of educational discussions.

Technologies Used

DBSCAN TF-IDF RegEx NLP Python Scikit-learn NLTK

Key Features

  • Advanced Text Preprocessing

    Engineered a comprehensive preprocessing pipeline using RegEx and TF-IDF for effective data cleaning and vectorization, ensuring optimal text representation for clustering.

  • Density-Based Clustering

    Adopted DBSCAN (Density-Based Spatial Clustering of Applications with Noise) to categorize student inquiries, which proved more effective than traditional k-means for identifying natural question groups.

  • Hyperparameter Optimization

    Improved cluster accuracy by 75% through careful adjustment of DBSCAN parameters, enabling better identification of core, border, and noise points in the question space.

  • Topic Summarization

    Implemented automated topic extraction for each cluster, providing instructors with a concise overview of the main themes in student questions.

Results and Impact

The system significantly improved the management of student inquiries by automatically grouping related questions together. This allowed instructors to address multiple questions with a single comprehensive response, reducing response time while improving answer quality. The clustering approach also revealed patterns in student confusion, helping instructors identify areas of the curriculum that needed additional clarification. Educational staff reported saving approximately 40% of the time previously spent on managing and responding to student questions.
