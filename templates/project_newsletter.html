Newsletter Management System

July 2024 - August 2024 Social Presence

Project Overview

This project involved developing an MVP (Minimum Viable Product) for Social Presence to automate the management and matching of newsletters with titles. The system incorporated ad detection, topic modeling, and sentiment analysis capabilities to process and organize email newsletters efficiently. This MVP streamlined the content management workflow and provided valuable insights into newsletter dynamics and advertising patterns.

Technologies Used

Python IMAP DFS Algorithm Selenium Fuzzy Matching Gemini API PostgreSQL NLP

Key Features

  • Email Content Retrieval

    Retrieved emails through IMAP, utilizing depth-first search (DFS) algorithms to parse HTML content and systematically store both metadata and content in a structured format.

  • Intelligent Ad Detection

    Employed fuzzy-matching techniques to detect advertisements by comparing content with a curated list of ad-disclosures, successfully identifying self-promotion and various ad types within newsletters.

  • Content Analysis

    Extracted ad URLs with Selenium for further analysis, and processed news content to identify topics, genres, and sentiment patterns using Gemini API's natural language processing capabilities.

  • Data Management

    Designed and implemented a PostgreSQL database schema to efficiently store and query the processed newsletter data, enabling quick access and comprehensive analysis.

Results and Impact

The MVP delivered to Social Presence significantly improved their newsletter processing workflow, reducing manual analysis time by approximately 70%. The automated ad detection achieved over 85% accuracy, while the topic modeling and sentiment analysis provided valuable insights into content performance. This system became a crucial tool for the company's content strategy, enabling data-driven decisions and more effective newsletter targeting.
